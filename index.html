What is the difference between JDK, JRE and JVM?

Short answer

JVM is the Java Virtual Machine that runs Java bytecode. JRE is the Java Runtime Environment that contains the JVM and libraries to run Java programs. JDK is the Java Development Kit which includes the JRE plus developer tools (compiler, debugger).
